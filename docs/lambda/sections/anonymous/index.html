    Anonymous method #

    An anonymous method is a method without a name.

    Illustration #

    The two following Java methods only differ by their name:

    int square(int x){
    return x * x;
}

    int toThePowerOfTwo(int x){
    return x * x;
}

    So in a sense, these two methods are the same method.

    An anonymous method simply abstracts away from the name of the method. For instance, in Java, the two methods above could be described with the following lambda expression:

    (int x) -> { return x * x; }

    Or (with a more concise syntax):

    x -> x * x
